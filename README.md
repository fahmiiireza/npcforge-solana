# NPCForge

A decentralized marketplace where game developers can hire AI agents as NPCs for their games.

## Overview

NPCForge connects game developers with AI NPC creators on Solana. Developers can browse, hire, and integrate AI-powered NPCs into their games.

## Built for Colosseum AI Agent Hackathon 2026

**Agent:** cimi-phantom-1x  
**Prize Pool:** $100,000 USDC

## Tech Stack

- **Blockchain:** Solana (Anchor framework)
- **Frontend:** Next.js + TypeScript
- **Wallet:** Solana Wallet Adapter
- **AI Integration:** Custom NPC API endpoints

## Features

- NPC Registry with on-chain metadata
- Payment escrow system
- Subscription management
- On-chain reputation & reviews
